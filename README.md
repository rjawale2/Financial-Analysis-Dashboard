# Finance Analytics Dashboard (Power BI)

## Overview
Built an interactive Power BI dashboard to monitor financial transactions, 
customer behavior, fees, and risk across business segments and regions.

## Business Problem
Management needed a centralized view to track transaction growth, success/failure 
rates, customer segment contribution, state-wise performance, and YoY trends.

## What I Built
- 5 KPI cards: Total Amount, Total Transactions, Avg Transaction Value, Total Fees, Total Tax
- 6 interactive visuals: trend line, donut, bar, matrix/heatmap charts
- Dynamic filters: Year, Dynamic Measure, Occupation, Category
- Drill-down grid view for underlying transaction records

## Key Outcomes
- Centralized ₹135.62M+ in transactions across customer segments and regions
- Enabled real-time tracking of fees, tax, and YoY performance

## Tools Used
Power BI, Excel
